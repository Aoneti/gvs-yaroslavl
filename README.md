# Отключение горячей воды в Ярославле 2026

Простая и удобная страница с актуальной информацией об отключениях горячего водоснабжения (ГВС) в Ярославле на 2026 год.

[![Open in browser](https://img.shields.io/badge/Открыть_страницу-2ea44f?style=for-the-badge&logo=html5&logoColor=white)](https://aoneti.github.io/gvs-yaroslavl/)

## Что это

Статическая HTML-страница, которая помогает жителям Ярославля быстро узнать график отключения горячей воды по своему адресу.

Основные возможности:
- Поиск по улице / адресу
- Быстрый просмотр дат и сроков отключения
- Минималистичный и мобильно-адаптивный дизайн
