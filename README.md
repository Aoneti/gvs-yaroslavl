# 🔥 Плановые отключения горячего водоснабжения в Ярославле

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![GitHub Pages](https://img.shields.io/badge/Hosted-GitHub_Pages-222222?logo=github)](https://aoneti.github.io/gvs-yaroslavl/)
[![Static Site](https://img.shields.io/badge/Type-Static_Website-brightgreen)](https://aoneti.github.io/gvs-yaroslavl/)

Статический веб‑сервис для поиска актуальной информации об отключениях горячего водоснабжения в Ярославле. Позволяет быстро найти адрес и узнать даты плановых отключений.

<div align="center">

### **🌐 [Открыть сайт](https://aoneti.github.io/gvs-yaroslavl/)**

</div>

## Возможности

- **Поиск** — поиск по улице и номеру дома с поддержкой частичных совпадений
- **Детальная информация** — отображение этапов работ: ремонт, гидравлические испытания, дата возобновления подачи воды
- **Интерактивная карта** — визуализация адресов отключений на карте, при клике на маркер открывается подсказка с датами
- **Адаптивный дизайн** — корректное отображение на мобильных устройствах, планшетах и десктопах

## Что внутри

| Файл | Описание |
|---|---|
| `index.html` | Главная страница — поиск по адресу |
| `map.html` | Интерактивная карта отключений |
| `gvs-core.js` | Общая логика для `index.html` и `map.html` |
| `app-index.js` | Логика главной страницы: поиск, фильтрация, пагинация |
| `app-map.js` | Логика карты: инициализация Leaflet, кластеризация маркеров |
| `ui-widgets.js` | UI-компоненты: модальные окна, уведомления, кнопки |
| `analytics.js` | Модуль аналитики: отслеживание событий и метрик |
| `theme-init.js` | Инициализация темы (светлая/тёмная) |
| `data.json` | База отключений: адреса, даты ремонта и гидравлических испытаний |
| `geo.json` | Координаты адресов |
| `styles.css` | Cтили интерфейса |
| `convert.ipynb` | Jupyter-ноутбук для предобработки исходных данных из Excel → `data.json` |
| `geocoder.ipynb` | Геокодирование адресов через Nominatim API → `geo.json` |

### Структура проекта

```
├── index.html              # Главная страница
├── map.html                # Страница карты
├── styles.css              # Основные стили
├── config.js               # Централизованная конфигурация приложения
├── gvs-core.js             # Общая бизнес-логика
├── app-index.js            # Логика главной страницы
├── app-map.js              # Логика карты
├── ui-widgets.js           # Виджеты
├── analytics.js            # Аналитика
├── theme-init.js           # Тема оформления
├── data.json               # Данные об отключениях
├── geo.json                # Геоданные
├── fonts/                  # Шрифты
├── vendor/                 # Сторонние библиотеки
│   ├── leaflet.js/css      # Leaflet
│   └── leaflet.markercluster.js/css  # MarkerCluster
└── convert.ipynb           # Конвертация данных из Excel → data.json
└── geocoder.ipynb          # Геокодирование адресов → geo.json
```

## Технологический стек

### Фронтенд
- **HTML5 / CSS / JavaScript** — вёрстка и логика
- **[Leaflet](https://leafletjs.com/)** + **[MarkerCluster](https://github.com/Leaflet/Leaflet.markercluster)** — интерактивная карта с кластеризацией

### Обработка данных
- **Python** + **pandas** — очистка и трансформация данных
- **Jupyter Notebook** — интерактивная разработка скриптов конвертации
- **Nominatim (OpenStreetMap)** — геокодирование адресов

### Инфраструктура
- **GitHub Pages** — хостинг статического контента
