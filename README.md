# Отключение горячей воды в Ярославле

Статичный веб-сайт для поиска информации об отключениях горячего водоснабжения в Ярославле. Позволяет жителям быстро найти свой адрес и узнать даты плановых отключений.

**→ [Открыть сайт](https://aoneti.github.io/gvs-yaroslavl/)**

## ✨ Возможности

- **Поиск по адресу** — мгновенный поиск по улице и номеру дома
- **Этапы отключений и даты подключений** — ремонтные работы, гидравлические испытания, дата возобновления подачи воды
- **Карта** — адреса перенесены на карту, при клике на маркер открывается попап с датами
- **Адаптивная вёрстка** — корректно отображается на мобильных устройствах
- **Быстрые подсказки** — кнопки с популярными улицами для удобного поиска

## Что внутри
 
| Файл | Описание |
|---|---|
| `index.html` | Главная страница — поиск по адресу с пагинацией |
| `map.html` | Интерактивная карта отключений (Leaflet + кластеризация) |
| `data.json` | База отключений: адреса, даты ремонта и гидравлических испытаний |
| `geo.json` | Координаты адресов, полученные через геокодирование |
| `gvs_convert.ipynb` | Colab-блокнот: конвертация Excel → `data.json` |
| `geocoder_v2.ipynb` | Colab-блокнот: геокодирование адресов → `geo.json` |

## Технологии
 
- Чистый HTML/CSS/JS — без фреймворков и сборщиков
- [Leaflet](https://leafletjs.com/) + [MarkerCluster](https://github.com/Leaflet/Leaflet.markercluster) — карта
- [Nominatim (OpenStreetMap)](https://nominatim.openstreetmap.org) — геокодирование
- Google Colab + pandas — обработка данных
- GitHub Pages — хостинг

## 📄 Лицензия

Проект распространяется свободно. Данные об отключениях являются общедоступной информацией коммунальных служб города Ярославля.
