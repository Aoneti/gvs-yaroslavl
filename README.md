# Отключение горячей воды в Ярославле

Простая и удобная страница с актуальной информацией об отключениях горячего водоснабжения (ГВС) в Ярославле.

## Основные возможности:
- Поиск по улице / адресу
- Быстрый просмотр дат и сроков отключения
- Минималистичный и мобильно-адаптивный дизайн
