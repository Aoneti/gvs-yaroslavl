# Скрипты обновления данных — ГВС-Ярославль

Два Google Colab-блокнота для подготовки данных сайта. Запускаются независимо друг от друга, но при полном обновлении используются последовательно.

---

## Общий порядок работы

```
Excel-файл отключений
        │
        ▼
 gvs_convert.ipynb   →   data.json
        │
        ▼
geocoder_v2.ipynb    →   geo.json
        │
        ▼
  Загрузить оба файла в репозиторий
```

---

## 1. `gvs_convert.ipynb` — Excel → data.json

Конвертирует таблицу отключений из Excel в `data.json`, который читает сайт.

**Что делает:**
- Читает все листы `.xlsx`-файла
- Нормализует адреса
- Удаляет дубликаты:
- Сохраняет результат в `data.json`

**Запуск:**

1. Откройте ноутбук в [Google Colab](https://colab.research.google.com)
2. Запустите ячейки по порядку

---

## 2. `geocoder_v2.ipynb` — геокодирование адресов

Берёт адреса из `data.json` в репозитории и получает для каждого координаты через [Nominatim (OpenStreetMap)](https://nominatim.openstreetmap.org). Результат сохраняется в `geo.json`.

**Что делает:**
- Загружает `data.json` напрямую с GitHub
- Для каждого уникального адреса пробует 4 стратегии запроса к Nominatim (от самой точной к приближённой)
- Валидирует каждый результат
- Кэширует результаты в `geo.json` после **каждого** адреса

**Запуск:**

1. Откройте ноутбук в [Google Colab](https://colab.research.google.com)
2. Запустите ячейки по порядку
3. Ячейка **шаг 7.1** опциональна: раскомментируйте её, если хотите продолжить прерванный сеанс (загрузите ранее сохранённый `geo.json`)
